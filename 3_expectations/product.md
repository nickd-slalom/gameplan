# Product Expectations

## Participation

- Hosted-game participation: 90% of games have at least one participant besides
  the host.

## Scheduling Confidence

- Convention schedules must prevent conflicts and double-bookings so attendees
  can commit to games with confidence.

## Host Preparedness

- Hosts must be able to see committed player counts before the convention
  starts so they can prepare copies, rule summaries, and materials.
