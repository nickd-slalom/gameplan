# Feature Intent: Convention Creation

## Metadata

- [Feature Name: Convention Creation]
- [Owner: Product]
- [Status: proposed]
- [Last Updated: 2026-06-10]

## Problem

Convention organizers need a way to define a convention before attendees and
hosts can plan around it. Without a shared convention record, scheduling,
capacity planning, and participation tracking cannot start.

## Desired Outcome

An organizer can create and edit a basic convention record that establishes the
event container required for later scheduling and participation workflows.

## Business Value

This establishes the event container that later scheduling features depend on
and gives organizers a reliable source of truth for planning and attendance
management.

## User Impact

- Organizers can publish the basic structure of an event.
- Organizers can edit a convention after it is created.
- Attendees and hosts can use the convention details to understand when and
  where scheduling applies.
- The product has event-level limits and time boundaries available from the
  beginning.

## Anti-Goals

- Not a full registration or ticketing system.
- Not support for multiple venues or complex room-level scheduling.
- Not convention search or discovery.
- Not social networking features.

## Unknowns

- None so far
